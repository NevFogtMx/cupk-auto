1、首先运行app.exe，运行后会生成config.json
2、退出app.exe，修改config.json内容（账号、密码）
3、再次运行app.exe
4、运行httpServe.exe，出现cmd窗口，显示Server started at http://localhost:8866/，点击地址进入主界面

后面运行可以使用run.bat，它会同时启动上面两个exe文件

--------------------------------------------------------------------------------------------------
每次修改密码后注意修改config.json内容。
